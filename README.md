# 연료 주입

## 기능 요구사항
1. 사전 세팅
    - 소나타(10KM/L) 2대, 아반떼(15KM/L) 1대, K5(13KM/L) 2대
    - 연료는 초기에 0L 이다.(주입되어 있는 연료량 없음)
    
2. 이동 거리를 입력 받는다

3. 각 차량별로 필요한 연료량을 나타낸다

## Car가 공통적으로 가져야 하는 기능
- 연비 (각 차량이 생성자에서 셋팅)
- 이동할 거리 (생성자의 인자로 받음)
- 필요한 연료를 계산하는 메소드 (추상메소드는 아님)

## 프로그래밍 요구사항
- 상속과 추상 메소드 활용
- if/else 사용하지 않음
